# Posts Directory

This directory will contain your blog post markdown files.

## Structure

Each post should be a markdown file with frontmatter:

```markdown
---
title: Your Post Title
date: Month Day, Year
description: Brief description of your post
tags: [tag1, tag2, tag3]
published: true
---

Your post content here...
```
